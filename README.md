# Windsurf 自动填卡

Chrome 浏览器扩展，用于在 Stripe 支付页面自动填写信用卡信息。

## 功能

- 批量导入信用卡
- 一键自动填卡
- 自动点击"开始试用"按钮
- 支持多种卡号格式

## 支持的卡号格式

```
卡号|月|年|CVC          # 6221487255344888|01|2033|761
卡号|月/年|CVC          # 6221487255344888|01/33|761
卡号|月/年(4位)|CVC     # 6221487255344888|01/2033|761
```

## 安装

1. 打开 Chrome，访问 `chrome://extensions/`
2. 开启 **开发者模式**
3. 点击 **加载已解压的扩展程序**
4. 选择本项目目录

## 使用

1. 点击扩展图标，批量添加信用卡
2. 打开 Stripe 绑卡链接
3. 点击页面上的 **一键填卡** 按钮
4. 等待自动填写完成并自动提交

## 版本

当前版本：1.0.6
